Generating Total Serialism Sheet Music

Project Date: Thanksgiving Break 2023

Project Summary

This project aims to either take in some structure for a total serialist piece of music or just a random seed and will create a piece of total serialist music. The project essentially implements a compiler for a simple (ish) total serialism "language" or takes a random seed and creates lilypond code. It then runs the lilypond compiler to make a PDF of the sheet music. What is Total Serialism?

Total Serialism is a type of music where everything is predetermined before writing Total Serialism generalizes on the idea of serialism and uses the rows to apply to rhythms, articulations and dynamics. Serialist style music (or 12-Tone) music uses all 12 tones in an equal temperment system (ie a piano) to make music. It is written by creating a row (denoted as P-0) and using the P-0 row to build a 12 tone matrix out of it. Using this matrix, one can use different transpositions or inversions of the row to make music. The P-0 row is chosen before writing the music and which combinations of inverted, transposed or reversed rows are used. In Serialist music the composer gets to choose the octave of the note and the rhythm as well as add their own dynamics or artiulations as they see fit.

However, in Total Serialism, Rhythms, Dynamics and Articulation are also predetermined by a row. This lends itself to a very algorithmic approach to music composition.
Total Serialism style music often is extremely unappealing to listen to and is very difficult to play due to the precision needed. However its algorithmic nature lends itself nicely for a programming project.

The Project

This project leverages the Lilypond compiler to turn a text file into sheet music through a shell script. There are two modes of generation: Randomly generated music or music generated off of the mini total serialist language. When the mode of music generation is random, all rows are randomly generated while when using a text input file they are from the user's input.

The program then creates the appropriate 12 tone matrices for each row then generates music from them. Each row is added to a text file in proper lilypond syntax.

The program is run through a shell script that invokes the lilypond compiler and opens the resulting sheet music.
